# Periodic-Table

Small App using HTML and CSS to make a Periodic-Table.

```css
:hover
:nth-child()
:nth-last-child()
transform: scale();
z-index: auto;
transition: 0.25s;
```

```html
<table>
  <tr>
    <td></td>
  </tr>
</table>  
```
